Launch a cluster¶

The first step is to spin up a Dask Cluster. In Coiled, this is done by creating a coiled.Cluster instance, there are several keyword arguments you can use to specify the details of your cluster further. Please read the cluster creation documentation to know more.

Note: that you will give a name to this cluster, if you don’t specify this keyword argument, clusters will be given a unique randomly generated name.

import coiled

The predictive analysis will use xgboost, so you can create a Coiled software environment using the following command in iPython or Jupyter.

# import coiled

import dask
x1, x2, y = taxi_full_wtips['fare_amount'], taxi_full_wtips['trip_distance'], taxi_full_wtips['tip_amount']

x1, x2, y = dask.persist(x1, x2, y)













Sx1 = x1.mean().compute()
Sx2 = x2.mean().compute()













Sx1x2 = (x1*x2).mean().compute()
Sx1x1 = (x1**2).mean().compute()
Sx2x2 = (x2**2).mean().compute()













import numpy as np
ZTZ = np.array([[1, Sx1, Sx2], 
                [Sx1, Sx1x1, Sx1x2],
                [Sx2, Sx1x2, Sx2x2]])













Sy = y.mean().compute()
Sx1y = (x1*y).mean().compute()
Sx2y = (x2*y).mean().compute()













ZTy = np.array([Sy, Sx1y, Sx2y])

A = np.linalg.solve(ZTZ, ZTy)
